# Student Management System in C

A simple console program to manage student information with file storage.

## Features
- Add students (ID, Name, Age)
- Show all students
- Data is saved in "students.txt" automatically
- Exit the program

## Skills learned
- Structures (struct)
- Arrays of structs
- Loops and conditions
- Functions
- File handling (read/write)
